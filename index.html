Membuat Aplikasi Web Ruby on Rails dari Nol

Belajar cara mengembangkan aplikasi web yang terhubung ke database menggunakan Ruby on Rails

Ruby merupakan bahasa pemrograman yang memiliki syntax yang elegan dan framework Ruby on Rails memungkinkan bagi developer untuk mengembangkan aplikasi web dengan cepat! Menguasai keduanya akan mewujudkan mimpimu menjadi seorang web developer.

Cara terbaik untuk belajar Ruby on Rails adalah dengan langsung belajar membuat aplikasi melalui metode studi kasus, dan dibimbing oleh developer yang sudah berpengalaman menggunakan Ruby on Rails selama bertahun-tahun.

Di online course ini, kita akan belajar secara mendalam dimulai dari bahasa Ruby, framework Ruby on Rails dan yang paling seru - studi kasus membuat web menggunakan Ruby on Rails.

Dengan memahami Ruby dan Ruby on Rails serta membuat langsung aplikasi dengan metode studi kasus, kamu akan mendapatkan pengalaman langsung cara membuat aplikasi web tidak hanya dari sisi how tapi juga dari sisi why.

Berikut ini topik-topik yang akan dipelajari melalui online course ini :

  • Pemrograman Bahasa Ruby
  • Object Oriented Programming di Ruby
  • MVC - Model View Controller
  • Controller, Request dan Response
  • Routes
  • RESTful Resources
  • Model dan Database
  • Migration
  • Active Record
  • Active Record Association
  • Forms
  • User Accounts
  • Authentication and Authorization
  • Best Practices, Tips, dan Clean code
  • Studi Kasus
